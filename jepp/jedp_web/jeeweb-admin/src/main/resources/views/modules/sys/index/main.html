JEEWEB是一款基于SpringBoot2.0+Spring+Mybatis+Mybatis Plus的敏捷开发系统；它是一款具有代码生成功能的智能快速开发平台；是以Spring Framework为核心容器，Spring MVC为模型视图控制器，Mybatis为数据访问层， Apache Shiro为权限授权层，Ehcahe对常用数据进行缓存，Disruptor作为并发框架，Bootstrap作为前端框架的优秀开源系统。
目前功能模块代码生成器、权限框架、数据字典、数据缓存、并发框架、数据监控、计划任务、多数据源管理、附件管理、UI模板标签、短信发送、邮件发送、统计功能等功能。

JEEWEB的开发方式采用（代码生成器快速设计生成代码->手工完善逻辑->丰富模板标签快速前端开发），可以快速协助java开发人员解决60%的重复工作，让开发人员更多关注业务逻辑的实现，框架使用前端模板标签，解放JAVA开发人员的开发压力，提高开发效率，为企业节省项目研发成本，减少开发周期。

Jeeweb适用范围

Jeeweb具有以下特点：

  1. SpringBoot+Spring+Mybatis+Hibernate+Shiro+ Ehcache+Disruptor+Jquery + Boostrap + Ztree等基础前后端架构架构
  2. 采用面向声明的开发模式， 基于泛型编写极少代码即可实现复杂的数据展示、数据编辑、表单处理等功能，在不使用代码生成器的情况下，也只需要很少的代码就能实现基础的CURD操作，再配合在线开发与代码生成器的使用，更加加快了开发的进度，将J2EE的开发效率成本提高，可以将代码减少60%以上。
  3. 在线开发(通过在线配置实现一个表模型的增删改查功能，无需一行代码，支持用户自定义表单布局)
  4. 代码生成器，支持多种数据模型,根据表生成对应的Entity,Service,Dao,Controller,JSP等,增删改查功能生成直接使用
  5. UI标签开发库，针对前端UI进行标准封装表，页面统一采用UI标签实现功能：数据datagrid,treegrid,FileInput,Editor,GridSelect等，实现JSP页面零JS，开发维护简洁高效
  6. 查询过滤器：只需前端配置，后台动态拼SQL追加查询条件；支持多种匹配方式（全匹配/模糊查询/包含查询/不匹配查询）
  7. 移动平台支持，对Bootstrap(兼容Html5)进行标准封装
  8. 灵活的权限控制，可控制到页面或按钮，满足绝大部分的权限需求,优化权限注解方便权限配置
  9. 完善的XSS防范及脚本过滤，彻底杜绝XSS攻击
  10. 支持分布式部署，session存储在redis中
  11. 友好的代码结构及注释，便于阅读及二次开发
  12. 引入quartz定时任务，可动态完成任务的添加、修改、删除、暂停、恢复及日志查看等功能
  13. 引入swagger文档支持，方便编写API接口文档
  14. 国际化（支持多语言，国际化的封装为多语言做了便捷支持）
  15. 多数据源（在线配置数据源，数据源工作类封装）
  16. 数据权限：整合Shiro权限
  17. 计划任务控制（在线配置计划任务、方便计划任务的时间调整规划）
  18. 邮件发送（配置邮件模版、邮件帐号的在线配置、邮件异步发送、邮件发送日志功能统计）
  19. 短信发送（配置短信模版、短信帐号的在线配置、短信异步发送、短信发送日志功能统计、支持短信发送平台动态切换）
  20. 多种首页风格切换,支持自定义首页风格。（Inspinia风格|ACE风格）
  21. 数据统计报表：丰富的报表统计功能
  22. 支持多种浏览器: Google, 火狐, IE,360 等
  23. 支持数据库: Mysql,Oracle10g,SqlServer等
  24. 基础权限: 用户，角色，菜单权限
  25. Web容器测试通过的有Jetty和Tomcat,Weblogic
  26. 要求JDK1.8+
JeeWeb敏捷开发平台，可以应用在任何JAVA WEB项目的开发中，是信息化建设的良好解决方案。 采用代码生成功能，以及手工完善逻辑代码的开发方式，可以显著提高开发效率60%以上，极大降低开发成本。
